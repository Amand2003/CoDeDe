# JavaScript IDE

A modern, browser-based IDE for writing and testing JavaScript code with real-time console output.

## Features

- **JavaScript Editor**: Write JavaScript code with syntax highlighting
- **Console Output**: See the results of your code execution in real-time
- **Real Browser Dialogs**: Support for alert(), confirm(), and prompt() functions
- **Error Handling**: Clear error messages with line number references
- **Dark Mode**: Toggle between light and dark themes for comfortable coding in any environment
- **Keyboard Shortcuts**: Boost your productivity with convenient shortcuts
- **Responsive Design**: Works on desktop and mobile devices

## How to Use

1. Open `index.html` in your web browser
2. Write JavaScript code in the Code Editor panel
3. Click "Run Code" or press Ctrl+Enter / Cmd+Enter to execute your code
4. See the output in the Console Output panel

### Dark Mode

- Use the toggle switch in the top-right corner to switch between light and dark modes
- Your preference will be saved for future sessions
- The IDE automatically detects your system's color scheme preference on first visit

### Keyboard Shortcuts

- **Ctrl+Enter / Cmd+Enter**: Run your code
- **Tab**: Insert spaces for indentation

### Supported JavaScript Features

The IDE supports all standard JavaScript features available in your browser, including:

- **Alert Dialogs**: Use `alert()` to show popup messages
- **Confirmation Dialogs**: Use `confirm()` to get user confirmation
- **Input Dialogs**: Use `prompt()` to get user input
- **Console Output**: Use `console.log()` to output data to the console

## Technologies Used

- HTML5
- CSS3
- JavaScript
- CodeMirror for code editing with syntax highlighting
- Font Awesome for icons

## Tips

- The console output shows both explicit `console.log()` calls and the return value of your code
- Error messages include the line number where the error occurred
- Dark mode is easier on the eyes when coding at night
- Visual feedback animations help confirm your actions
